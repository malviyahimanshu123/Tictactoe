# Tictactoe
A simple and interactive Tic Tac Toe game developed using HTML, CSS, and JavaScript. It enables two players to play the classic game in a clean and user-friendly interface. The game updates in real time as players take turns marking X and O on a 3×3 grid. It includes logic to check for winning conditions across rows, columns, and diagonals, along with draw detection when all cells are filled. A reset feature allows players to restart the game instantly. This project is beginner-friendly and helps in practicing core concepts like JavaScript logic building, event handling, and DOM manipulation.
